Software Name Similarity Ratios

  • Normal Ratio: Measures overall character similarity between two software names, considering exact character matches.
  • Token Ratio: Compares words in any order, ignoring their positions. Useful for names with rearranged words.
  • Partial Ratio: Checks if one name appears as a substring of the other. Detects partial overlaps.

Examples

  • AlphaSoft vs Soft Alpha: Normal ≈ 67%, Token ≈ 100%, Partial ≈ 80%
  • DataAnalyzer Pro vs Analyzer Pro: Normal ≈ 88%, Token ≈ 100%, Partial ≈ 100%
  • QuickEdit vs QuickEditor: Normal ≈ 91%, Token ≈ 91%, Partial ≈ 100%

Notes:

Setting a slider to 0 ignores that ratio. Adjusting sliders updates the comparison results in real time.
